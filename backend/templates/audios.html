Searching


By default, searching will search in tags, it's name, and audioid.

You can search for multiple terms by seperating terms by commas. I.E: lyrics, rock will search for songs with lyrics or rock in the tags or name

If you want to search for everything WITHOUT a term, put an exclamation (!) in-front of the term. I.E: !lyrics will search for songs without lyrics in the tags or name

If you want to search a specific field, use an equals (=) sign. I.E: tags=metal will search for anything with the tag 'metal'

Valid fields include:

  • ID
  • Name
  • Tags
  • Favorite
